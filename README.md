Queue Implementation Using Array in C

This repository contains a C program that implements a Queue data structure using an array.
The program demonstrates basic queue operations such as enqueue, dequeue, and peek (front element).

📌 Description

A queue is a linear data structure that follows the FIFO (First In, First Out) principle.
This implementation uses a fixed-size array and maintains two indices (front and rear) to manage queue operations.

🧠 How the Program Works

The queue is initialized with front and rear set to -1.

Enqueue:

Adds an element at the rear of the queue

Checks for overflow before insertion

Dequeue:

Removes and returns the front element

Checks for underflow before deletion

Front Element:

Displays the element at the front without removing it

🛠️ Features

Queue implementation using array

Enqueue, dequeue, and front (peek) operations

Overflow and underflow handling

Simple and beginner-friendly C code
